## Email Password  Authentication Extension

The Krista *'Email Password'* Authentication Extension supports authentication process for Krista workspace users in Krista Studio, Krista Client, and Krista Mobile.

You can configure this extension on the workspace Setup page under the Authentication tab.

After the set-up, authentication for workspace users will be handled through this extension.
