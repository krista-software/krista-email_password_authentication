# Use Case Examples

The following use cases and respective examples are critically important for this extension.

**1. Normal Login**

   Scenario: A user wants to access his/her online banking account. 

   Resulting Use Case: The user securely logs in with the username and password.


**2. Forgot Password**

   Scenario: A user forgets password.

   Resulting Use Case: The user resets the password via email after clicking the "Forgot Password" link.


**3. Change Password**

   Scenario: A user wants to update the password.

   Resulting Use Case: The user securely changes the password within the account.


**4. Account Blocking**

   Scenario: Unauthorized attempts to access a user's account.

   Resulting Use Case: The system temporarily blocks the account after five unsuccessful login attempts.


**5. Account Recovery**

   Scenario: A user needs to recover the account after forgetting password.

   Resulting Use Case: The user regains account access by clicking a password reset link received via email.
